Factorization domain package.

This package contains classes for solvable polynomials rings as (non-unique) factorization domains. Methods provided with interface GreatestCommonDivisor are e.g. greatest common divisors gcd(), primitive part primitivePart() or coPrime().

The implementation is new research.
